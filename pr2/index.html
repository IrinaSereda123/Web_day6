📖 Часть 1. Что такое DOM?

DOM (Document Object Model) — это представление HTML-документа в виде дерева тегов. Каждый тег является объектом, вложенные теги — «детьми» родительского элемента, а текст внутри тега — текстовым узлом.

🔍 Как посмотреть DOM: Нажми F12 → вкладка Elements. Там видно всё дерево страницы. Можно сворачивать/разворачивать теги и смотреть стили.

🌱 Пример DOM-дерева

Для такого HTML:

<!DOCTYPE HTML>
<html>
<head>
    <title>О лосях</title>
</head>
<body>
    Правда о лосях.
</body>
</html>

DOM-дерево будет выглядеть так:

html
├── head
│   └── title
│       └── "О лосях"
└── body
    └── "Правда о лосях."
💡 Важно знать:
  • Узлы-элементы — HTML-теги, основные строительные блоки.
  • Текстовые узлы — содержат текст (обозначаются как #text).
  • Комментарии — тоже являются узлами DOM.
  • Пробелы и переводы строк — тоже текстовые узлы!
  • Существует 12 типов узлов, но на практике мы работаем с 4: document, элементы, текстовые узлы, комментарии.

🧩 Задание 1. Дочерние элементы DOM

Для страницы с таким кодом напиши JavaScript, который получит доступ к указанным элементам.

<html>
<body>
    <div>Пользователи:</div>
    <ul>
        <li>Джон</li>
        <li>Пит</li>
    </ul>
</body>
</html>

✏️ Твоя задача:

  1. Найти элемент <div>
  2. Найти элемент <ul>
  3. Найти второй <li> (с именем Пит)

Напиши код в консоли браузера или в теге <script>.

💡 Подсказки:
  • document.body.firstElementChild — первый дочерний элемент body
  • document.body.children[0] — тоже первый дочерний элемент
  • document.body.lastElementChild — последний дочерний элемент body
  • element.lastElementChild — последний дочерний элемент
🔑 Посмотреть решение (только если совсем не получается) 
// DOM-узел элемента <div>:
document.body.firstElementChild
// или
document.body.children[0]

// DOM-узел элемента <ul>:
document.body.lastElementChild
// или
document.body.children[1]

// Второй <li> (с именем Пит):
document.body.lastElementChild.lastElementChild

🎲 Задание 2. Выделение ячеек по диагонали

Напиши код, который выделит красным цветом все ячейки в таблице по диагонали.

✏️ Твоя задача:

Создай HTML-страницу с таблицей 5×5 и кнопкой. При нажатии на кнопку ячейки на главной диагонали должны стать красными.

Используй document.getElementById и свойство rows.

💡 Подсказки:
  • document.getElementById('myTable') — получаем таблицу
  • table.rows — коллекция всех строк таблицы
  • row.cells — коллекция ячеек в строке
  • На диагонали индекс строки равен индексу ячейки: table.rows[i].cells[i]
🔑 Посмотреть решение (только если совсем не получается) 
function highlightDiagonal() {
    const table = document.getElementById('myTable');
    for (let i = 0; i < table.rows.length; i++) {
        table.rows[i].cells[i].style.backgroundColor = 'red';
        table.rows[i].cells[i].style.color = 'white';
    }
}

🔍 Задание 3. Использование getElementById

Если у элемента есть атрибут id, мы можем получить его вызовом document.getElementById(id), где бы он ни находился.

✏️ Твоя задача:

Создай страницу с тремя элементами, у которых есть id:

  • Заголовок <h1 id="title">Привет!</h1>
  • Параграф <p id="text">Какой-то текст</p>
  • Кнопка <button id="myButton">Нажми меня</button>

С помощью getElementById измени:

  1. Цвет заголовка на синий
  2. Размер шрифта параграфа на 20px
  3. Фон кнопки на зелёный
🔑 Посмотреть решение (только если совсем не получается) 
document.getElementById('title').style.color = 'blue';
document.getElementById('text').style.fontSize = '20px';
document.getElementById('myButton').style.background = 'green';

🔑 Посмотреть решение (только если совсем не получается) 
const lights = document.querySelectorAll('.light');
const states = [
    [1, 0, 0],  // красный
    [1, 1, 0],  // красный + жёлтый
    [0, 0, 1],  // зелёный
    [0, 1, 0]   // жёлтый
];
let stateIndex = 0;

function updateLights() {
    for (let i = 0; i < lights.length; i++) {
        if (states[stateIndex][i]) {
            lights[i].classList.add('active');
        } else {
            lights[i].classList.remove('active');
        }
    }
    stateIndex = (stateIndex + 1) % states.length;
}

updateLights();
document.getElementById('nextLight').onclick = updateLights;

🔎 Бонус-задание 2. Исследователь страницы

Открой любую страницу в интернете (например, google.com, wikipedia.org или любую другую). Нажми F12 и перейди на вкладку Elements.

📝 Что нужно сделать:
  • 🔍 Найди тег <body> — что у него внутри?
  • 🔍 Найди тег <title> — какой текст в нём?
  • 🔍 Найди элемент с классом header или menu (если есть)
  • ✏️ Попробуй изменить текст любого элемента прямо в браузере (двойной клик по тексту во вкладке Elements)
  • 🎨 В правой панели (Styles) измени цвет фона любого элемента
  • 🖱️ Нажми на кнопку Выбрать элемент (значок стрелки в левом верхнем углу инспектора) и наведи на любой элемент страницы — браузер сам найдёт его в DOM-дереве

Результат: напиши в тетради (или сохрани в файл), какие элементы ты нашёл и какие изменения сделал.

💡 Инструменты разработчика:
  • Styles — здесь видно CSS, применённый к элементу. Почти всё можно отредактировать на месте.
  • Computed — итоговые CSS-свойства элемента.
  • Event Listeners — обработчики событий, привязанные к элементу.
  • Console — можно вводить JavaScript-код прямо здесь. Выбранный элемент доступен как $0.

📚 Шпаргалка по DOM

Команда Что делает
document.getElementById('id') Находит элемент по id
document.getElementsByClassName('class') Находит все элементы по классу (коллекция)
document.querySelector('.class') Находит первый элемент по CSS-селектору
document.querySelectorAll('div') Находит ВСЕ элементы по CSS-селектору
element.parentElement Родитель элемента
element.children Дети элемента
element.firstElementChild Первый дочерний элемент
element.lastElementChild Последний дочерний элемент
element.classList.add('class') Добавляет класс
element.classList.remove('class') Удаляет класс
element.classList.toggle('class') Переключает класс (добавляет/удаляет)
element.style.color = 'red' Меняет стиль
element.innerHTML = 'текст' Меняет HTML-содержимое
element.textContent = 'текст' Меняет текстовое содержимое
💻 Взаимодействие с консолью:
  • На вкладке Elements выбери любой элемент
  • Нажми Esc — откроется консоль
  • Выбранный элемент доступен как $0
  • Попробуй: $0.style.background = 'red'
